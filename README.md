# CPAanalysis

This repository contains raw survey data and a generated visual summary for key findings.

## Visual deliverable

- `visuals/cpa_survey_key_findings.svg`: high-quality infographic-style visualization of five core survey questions.
- `scripts/generate_survey_visual.py`: reproducible generator script that parses the CSV export and writes the SVG visual.

## Regenerate the visual

```bash
python scripts/generate_survey_visual.py
```
